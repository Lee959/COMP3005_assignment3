
## Step to Compile and Run the Project
1. **Clone the Repository**  
   ```bash
   git clone 'https://github.com/Lee959/COMP3005_assignment3.git'
    ```
2. **Open the Project Directory**  
   ```bash
   cd <directory-file-path-Java-source-file-is-located>
   ```
3. **Compile the Java Files**  
   ```bash
    javac *.java
    ```
4. **Run the Main Class**
    ```bash
   java ConnectStudentDatabase
   ```
**Note:**: Or if you have ide like IntelliJ or Eclipse, you can import the project and run the `ConnectStudentDatabase` class directly from the IDE.
